# 2026-03-23 心跳保护与电流控制驱动总结

## 1. 目标
- 基于已完成的 ModBus 协议封装，补充“具体控制”能力。
- 实现心跳保护机制接口。
- 实现电流控制接口（目标电流 + 模式切换）。
- 驱动器 ID 使用宏配置，默认 1。

## 2. 新增文件

### 2.1 motor_modbus.h
文件: Core/Inc/motor_modbus.h
- 新增从站地址宏:
  - `MOTOR_MODBUS_SLAVE_ID`（默认 1）
- 新增寄存器地址宏:
  - 心跳寄存器 `6000`
  - 控制模式寄存器 `6001`
  - 目标电流寄存器 `6002`
- 新增电流控制模式宏:
  - `MOTOR_MODBUS_MODE_CURRENT_CONTROL` = 0
- 提供接口:
  - `motor_modbus_init`
  - `motor_modbus_write_heartbeat`
  - `motor_modbus_heartbeat_tick`
  - `motor_modbus_set_current_mode`
  - `motor_modbus_set_target_current_10ma`
  - `motor_modbus_set_target_current_ma`
  - `motor_modbus_set_current_control_10ma`

### 2.2 motor_modbus.c
文件: Core/Src/motor_modbus.c
- 通过 `modbus_rtu_write_single_register` 实现上层业务写寄存器。
- `motor_modbus_heartbeat_tick` 内部递增心跳值并写入寄存器 6000。
- `motor_modbus_set_current_control_10ma` 按文档流程执行:
  1. 先写目标电流寄存器 6002
  2. 再写控制模式寄存器 6001 = 0（电流控制）
- 支持 mA 到 10mA 的换算接口，并做边界检查。

## 3. main.c 接入
文件: Core/Src/main.c
- 引入 `motor_modbus.h`。
- 初始化阶段调用 `motor_modbus_init()`。
- 在主循环用户区增加注释示例:
  - 心跳周期更新
  - 电流控制 1A 示例（100 * 10mA）

## 4. 使用说明

### 4.1 修改驱动器 ID
在 `Core/Inc/motor_modbus.h` 中修改:

```c
#define MOTOR_MODBUS_SLAVE_ID (1U)
```

### 4.2 心跳机制
- 建议周期约为驱动器心跳超时时间的 1/2。
- 可周期调用:
  - `motor_modbus_heartbeat_tick(timeout_ms)`

### 4.3 电流控制
- 设置目标电流并切换到电流模式:
  - `motor_modbus_set_current_control_10ma(current_10ma, timeout_ms)`
- 例如:
  - 1A -> `100`
  - -1A -> `-100`

## 5. 备注
- 当前实现基于阻塞式 UART（HAL）。
- 若后续需要提升实时性，可扩展为中断或 DMA。
