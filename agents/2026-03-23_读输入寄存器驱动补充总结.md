# 读输入寄存器驱动补充总结

**日期**: 2026-03-23

## 改动背景

需要读取驱动器的故障信息(5000)和实时转速(5001-5002)。这些属于**输入寄存器**，需要使用 ModBus 功能码 **0x04**（读输入寄存器），而非之前已实现的 0x03（读保持寄存器）。

## 改动内容

### 1. 协议层 — modbus_rtu

**modbus_rtu.h**:
- 新增功能码宏 `MODBUS_FUNC_READ_INPUT_REGISTERS (0x04U)`
- 新增函数声明 `modbus_rtu_read_input_registers`

**modbus_rtu.c**:
- 新增 `modbus_rtu_read_input_registers` 实现，报文格式与 0x03 一致，仅功能码为 0x04

### 2. 业务层 — motor_modbus

**motor_modbus.h**:
- 新增寄存器别名宏:
  - `MOTOR_MODBUS_REG_FAULT_INFO` → 5000
  - `MOTOR_MODBUS_REG_REALTIME_SPEED_H` → 5001
- 新增函数声明:
  - `motor_modbus_read_fault_info`
  - `motor_modbus_read_speed_erpm`

**motor_modbus.c**:
- `motor_modbus_read_fault_info`: 读取故障信息寄存器(5000)，返回 uint16_t 故障码
- `motor_modbus_read_speed_erpm`: 读取实时转速寄存器(5001-5002)，组合为 int32_t erpm 值

## 帧格式对照

| 功能 | 发送帧 | 说明 |
|---|---|---|
| 读故障信息 | `01 04 13 88 00 01 CRC` | 功能码 0x04, 地址 5000, 数量 1 |
| 读实时转速 | `01 04 13 89 00 02 CRC` | 功能码 0x04, 地址 5001, 数量 2 |

## 使用示例

```c
uint16_t fault = 0U;
int32_t speed_erpm = 0;

motor_modbus_read_fault_info(&fault, 300U);
motor_modbus_read_speed_erpm(&speed_erpm, 300U);

/* erpm 转 rpm: rpm = speed_erpm / MOTOR_MODBUS_POLE_PAIRS */
```
