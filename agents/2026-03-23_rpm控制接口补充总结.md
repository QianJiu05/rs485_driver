# 2026-03-23 rpm控制接口补充总结

## 1. 修改目标
- 提供直接基于 rpm 的控制接口。
- 内部自动完成 rpm -> erpm 换算，减少上层业务负担。

## 2. 头文件新增接口
文件: Core/Inc/motor_modbus.h

- `motor_modbus_set_target_speed_rpm(int32_t speed_rpm, uint16_t pole_pairs, uint32_t timeout_ms)`
- `motor_modbus_set_speed_control_rpm(int32_t speed_rpm, uint16_t pole_pairs, uint32_t timeout_ms)`

## 3. 源文件实现
文件: Core/Src/motor_modbus.c

- 新增 rpm 换算逻辑:
  - `erpm = rpm * pole_pairs`
- 新增参数校验:
  - `pole_pairs` 不能为 0
  - 换算结果不能超出 `int32_t` 范围
- 新增流程接口:
  - 先设置目标 rpm（内部换算并写 6003-6004）
  - 再切换到转速模式（6001=1）

## 4. main 示例补充
文件: Core/Src/main.c

- 新增注释示例调用:
  - `motor_modbus_set_speed_control_rpm(1000, 4U, 100U)`

## 5. 结果
- 现在可直接按机械转速 rpm 发控制命令。
- 无需上层手动做 erpm 换算。
