# 2026-03-23 转速控制驱动补充总结

## 1. 修改目标
- 根据驱动文档补充转速控制能力。
- 支持目标转速写入、转速模式切换、速度环加减速度设置。

## 2. 协议层扩展

### 2.1 文件
- Core/Inc/modbus_rtu.h
- Core/Src/modbus_rtu.c

### 2.2 新增内容
- 新增功能码宏: `MODBUS_FUNC_WRITE_MULTIPLE_REGISTERS (0x10)`。
- 新增接口:
  - `modbus_rtu_write_multiple_registers(...)`
- 实现细节:
  - 按 ModBus 大端格式打包多个寄存器。
  - 校验响应中的起始地址与寄存器数量。

## 3. 业务层扩展

### 3.1 文件
- Core/Inc/motor_modbus.h
- Core/Src/motor_modbus.c

### 3.2 新增宏
- `MOTOR_MODBUS_REG_TARGET_SPEED_H` -> 6003
- `MOTOR_MODBUS_REG_SPEED_LOOP_ACCEL_H` -> 6016
- `MOTOR_MODBUS_REG_SPEED_LOOP_DECEL_H` -> 6025
- `MOTOR_MODBUS_MODE_SPEED_CONTROL` -> 1

### 3.3 新增接口
- `motor_modbus_set_speed_mode(timeout_ms)`
- `motor_modbus_set_target_speed_erpm(speed_erpm, timeout_ms)`
- `motor_modbus_set_speed_loop_accel_erpm_s(accel_erpm_s, timeout_ms)`
- `motor_modbus_set_speed_loop_decel_erpm_s(decel_erpm_s, timeout_ms)`
- `motor_modbus_set_speed_control_erpm(speed_erpm, timeout_ms)`

### 3.4 实现说明
- 新增内部辅助函数 `motor_modbus_write_i32(...)`，将 `int32_t` 拆分为高/低16位并一次写入两个寄存器。
- 转速控制流程按文档执行:
  1. 先写目标转速（6003-6004）
  2. 再写控制模式=1（6001）

## 4. 示例更新
- 文件: Core/Src/main.c
- 在注释示例中增加:
  - 速度环加速度设置（6016-6017）
  - 速度环减速度设置（6025-6026）
  - 目标转速+模式切换的一体调用

## 5. 结果
- 在原有心跳与电流控制基础上，已可直接调用转速控制接口。
- 新增函数均包含中文注释（功能、参数、返回值）。
