# 2026-03-23 磁极对数宏定义调整总结

## 1. 修改目标
- 将 rpm 控制示例中的磁极对数从硬编码改为宏定义。

## 2. 修改内容

### 2.1 新增宏定义
文件: Core/Inc/motor_modbus.h
- 新增宏:
  - `MOTOR_MODBUS_POLE_PAIRS`，默认值 `4U`
- 作用:
  - 统一配置电机磁极对数，用于 rpm 到 erpm 的换算场景。

### 2.2 示例调用替换
文件: Core/Src/main.c
- 将:
  - `motor_modbus_set_speed_control_rpm(1000, 4U, 100U)`
- 替换为:
  - `motor_modbus_set_speed_control_rpm(1000, MOTOR_MODBUS_POLE_PAIRS, 100U)`

## 3. 结果
- 消除硬编码，后续只需修改一处宏即可适配不同电机。
